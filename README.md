# Koh Larn Smart Eco Island — CMS (หลังบ้าน)

ระบบจัดการหลังบ้านด้วย **Nest.js** เสิร์ฟหน้าเว็บในตัว (Handlebars) ใช้ธีมสีโทนเดียวกับแอปมือถือ
ข้อมูลเป็น **mockup ในหน่วยความจำ** (รีสตาร์ตแล้วกลับเป็นค่าเริ่มต้น) — ยังไม่ผูกฐานข้อมูลจริง

## 2 โหมดการใช้งาน

**ผู้ประกอบการ (Operator)**
- เพิ่ม/แก้ไข โรงแรม ร้านอาหาร หรือสถานที่ท่องเที่ยว
- ทุกรายการที่ส่ง จะมีสถานะ "รออนุมัติ" จนกว่าเจ้าหน้าที่จะอนุมัติ จึงจะแสดงบนแอป

**เจ้าหน้าที่ (Officer)** — ต้องใส่ **รหัสเจ้าหน้าที่** ก่อนเข้าใช้งาน
- อนุมัติ / ไม่อนุมัติ (พร้อมเหตุผล) คำขอลงสถานที่จากผู้ประกอบการ
- ดูสถานที่ทั้งหมดในระบบ
- จัดการข่าวสาร: เพิ่ม / แก้ไข / ลบ / เผยแพร่หรือเก็บเป็นฉบับร่าง

## การติดตั้งและรัน

```bash
cd koh-larn-cms
npm install
npm start          # รันที่ http://localhost:3000
```

โหมด dev (auto-reload): `npm run dev`
Build เป็น JS: `npm run build` แล้ว `npm run start:prod`

## บัญชีสำหรับเดโม

| โหมด | วิธีเข้า |
|------|---------|
| ผู้ประกอบการ | อีเมล **`seaview@kohlarn.app`** / รหัสผ่าน **`123456`** หรือกด "สมัครสมาชิก" เพื่อสร้างบัญชีใหม่ |
| เจ้าหน้าที่ | Username **`admin`** / Password **`admin1234`** (อีกบัญชี: `pr` / `pr123456`) |

ผู้ประกอบการสมัครสมาชิกได้เองที่หน้า `/register` (กรอกชื่อกิจการ อีเมล รหัสผ่าน) — บัญชีใหม่จะล็อกอินเข้าใช้งานได้ทันที
บัญชีเจ้าหน้าที่กำหนดไว้ในไฟล์ `src/data/store.ts` (`officers`)

เปลี่ยนรหัสเจ้าหน้าที่ได้ผ่าน environment variable:
```bash
OFFICER_CODE=your-code npm start
```

## Public API (ให้แอปมือถือดึงข้อมูล)

| Endpoint | ข้อมูล |
|----------|--------|
| `GET /api/places` | สถานที่ที่ "อนุมัติแล้ว" เท่านั้น |
| `GET /api/news` | ข่าวที่ "เผยแพร่แล้ว" เท่านั้น |

## โครงสร้าง

```
src/
  main.ts                 bootstrap (hbs + session + static)
  app.module.ts           รวม controller + middleware
  data/store.ts           mock data + helpers (in-memory)
  auth/                   login 2 โหมด + รหัสเจ้าหน้าที่ + guard middleware
  operator/               โหมดผู้ประกอบการ
  officer/                โหมดเจ้าหน้าที่ (อนุมัติ + ข่าวสาร)
  api/                    public API สำหรับแอป
views/                    หน้าเว็บ (Handlebars)
public/styles.css         ธีมโทนเดียวกับแอป
```

> หมายเหตุ: เป็น mockup สำหรับนำเสนอ flow การทำงาน เมื่อพร้อมใช้งานจริง สามารถต่อ TypeORM/Prisma + PostgreSQL,
> เพิ่ม hashing รหัสผ่าน และอัปโหลดไฟล์รูปจริงได้
```
